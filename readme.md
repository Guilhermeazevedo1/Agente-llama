## Seu Assistente Virtual Multi-LLM (Projeto Simples de Estudo)
Este é um projeto simples de chatbot construído com Streamlit e LangChain, desenvolvido exclusivamente para fins de estudo e aprendizado.

O objetivo é criar uma interface básica de chat que demonstre a modularidade de conectar-se e alternar entre diferentes Grandes Modelos de Linguagem (LLMs), incluindo Hugging Face Hub (Llama 3), OpenAI e Ollama (modelos locais).

## Objetivo e Foco Principal
O foco principal deste projeto de estudo é demonstrar:

- Interface Simples: Construção de um chat funcional usando a estrutura básica do Streamlit.

- Modularidade de LLMs: Configuração de diferentes provedores de LLM (hf_hub, openai, ollama) e a capacidade de alternar entre eles facilmente via código.

- Gerenciamento de Conversa: Implementação do Histórico de Chat (chat_history) utilizando os componentes de mensagens do LangChain.

- Prompt Engineering: Aplicação correta de templates de prompt para modelos específicos (ex: formatação de chat para o Llama 3).
